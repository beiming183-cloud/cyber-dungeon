## 🎮 游戏简介

《赛博地牢》是一款像素风格的肉鸽类（Roguelike）动作游戏。玩家需要在随机生成的地牢中生存，击败不断涌来的敌人，收集资源，升级技能，挑战强大的Boss，并探索多个地图。

### 游戏特色
- 🎯 **5种独特角色**：每个角色都有不同的属性和技能组合
- ⚔️ **丰富的技能系统**：主动技能 + 被动技能双重成长
- 🛒 **商店系统**：使用金币强化角色属性和技能
- 🗺️ **多地图系统**：每个地图难度递增，充满挑战
- 💎 **路线天赋**：战斗、生存与探索天赋提供明确的开局构筑
- 🎨 **炫酷特效**：华丽的视觉特效和拖尾效果

## 💻 系统要求

- **操作系统**：Windows 7/8/10/11
- **内存**：至少 2GB RAM
- **显卡**：支持 OpenGL 2.0
- **存储空间**：约 100MB
- **字体**：需要支持中文字体（Windows 系统自带）

## 🎮 基本操作

### 移动控制
- **W / ↑**：向上移动
- **S / ↓**：向下移动
- **A / ←**：向左移动
- **D / →**：向右移动
- **支持中文输入法**：WASD 键在中文输入法下也能正常移动

### 攻击
- **鼠标左键**：普通攻击（向鼠标位置方向）
- **数字键 1-5**：释放主动技能

### 界面操作
- **鼠标点击**：选择角色和天赋
- **← / → 或 A / D**：切换角色、天赋和技能构筑选项
- **ESC**：返回上一级菜单
- **空格键 / Enter**：确认选择、开始游戏
- **数字键 1-3**：在选择界面快速选择

## 🎯 天赋系统

选择角色后，需要选择一种初始天赋。三种路线会给予固定的核心能力，开局方向更清晰：

### ⚔️ 战斗天赋
- **效果**：自动攻击 Lv.1，全局伤害 +8%
- **适合**：追求输出的玩家

### 🛡️ 生存天赋
- **效果**：
  - 防护盾 Lv.1
  - 最大生命 +15%
  - 受到伤害减少 **12%**
- **适合**：新手或追求生存的玩家

### 💰 探索天赋
- **效果**：
  - 环绕光球 Lv.1
  - 额外探索宝箱 +1
  - 金币收益 +25%
  - 经验值 +10%
- **适合**：喜欢快速发展的玩家

## 🎪 技能系统

### 主动技能
- **数量**：每个角色有 5 个主动技能
- **使用**：按数字键 1-5 释放
- **冷却时间**：每个技能都有独立的冷却时间
- **显示**：技能栏显示在屏幕底部，可以看到冷却状态

### 被动技能

#### 如何获得被动技能？
- **初始天赋**：按路线获得固定的核心被动技能
- **构筑能量**：探索宝箱、击败精英、连续完成波次或击败 Boss 可以获得能量
- **三选一升级**：能量积满后选择升级现有技能或获得新技能，后续升级所需能量逐步增加
- **限制**：最多同时拥有 **5个被动技能**

#### 被动技能列表

| 技能名称 | 图标 | 描述 | 最高等级 |
|---------|------|------|---------|
| **环绕光球** | ⚪ | 围绕自身旋转的光球攻击敌人 | 5 |
| **防护盾** | 🛡️ | 减少受到的伤害 | 5 |
| **自动攻击** | 🎯 | 自动向敌人发射攻击 | 5 |
| **连锁闪电** | ⚡ | 发射闪电球，在敌人之间弹跳 | 5 |
| **冰冻地雷** | 💣 | 放置地雷，爆炸时冻结敌人 | 5 |
| **吸血飞刃** | 🗡️ | 投掷回旋飞刃，造成伤害并回血 | 5 |
| **分身幻影** | 👥 | 创造分身吸引敌人，5秒后爆炸 | 5 |
| **重力领域** | 🌀 | 自动在敌人位置创造重力场 | 5 |
| **弹性防护罩** | 🛡️ | 间断性出现的防护罩，可以将怪弹开 | 5 |
| **无敌之心** | ❤️ | 每级增加一条命（最多2级） | 2 |

## 🛒 商店系统

### 如何打开商店？
1. 击败 **5个金色发光的小Boss**（精英怪）
2. Boss 会掉落一个**宝箱** 💎
3. 走到宝箱上拾取
4. 按 **B** 键打开商店

### 商店功能

| 商品 | 价格 | 效果 | 说明 |
|------|------|------|------|
| **增加生命上限** | 1000金币 | +20 最大生命值 | 永久提升 |
| **增加防御力** | 1000金币 | +5 防御力 | 永久提升 |
| **减少技能CD** | 1000金币 | 所有技能CD -0.1秒 | 可重复购买 |
| **增加投射物数量** | 1000金币 | 所有技能投射物 +1 | 可重复购买 |

### 商店操作
- **↑/W / ↓/S**：选择商品
- **回车键**：购买选中商品
- **B 键或 ESC**：关闭商店

### 重要提示
- ⚠️ 商店升级会在**每张新地图重置**
- 💰 商品价格较高（至少 1000 金币），需要合理规划
- 🎯 建议优先升级适合自己角色类型的属性

## 🗺️ 游戏机制

### 地图系统
- **随机生成**：每局游戏地图都是随机生成的
- **探索目标**：开局会显示最近能量宝箱的方向、距离和剩余数量
- **宝箱奖励**：探索宝箱轮换提供金币、恢复或构筑能量
- **波次节奏**：第一波只有 3 个敌人，清场后有 5 秒探索与观察时间
- **难度递增**：敌人数量和强度随波次平缓提升

### 敌人类型

| 敌人 | 特点 |
|------|------|
| 👹 **哥布林** | 基础敌人，平衡型 |
| 🦠 **史莱姆** | 速度慢，血量高 |
| 🦇 **蝙蝠** | 速度快，血量低 |
| 💀 **骷髅** | 平衡型，伤害较高 |
| 😈 **恶魔** | 强力敌人，伤害高 |
| 👻 **幽灵** | 速度极快 |
| 🕷️ **蜘蛛** | 平衡型 |

### Boss 战
- **出现条件**：每 **5波** 出现一次
- **Boss类型**：深渊龙王
- **特点**：
  - 更大的形态、独立全局血条与阶段光环
  - 按剩余生命切换预判追击、环绕压迫和狂暴冲刺三种阶段
  - 击败后掉落 Boss 战利品宝箱和大量构筑能量

### 精英怪（金色Boss）
- **外观**：金色发光，头顶皇冠 👑
- **特点**：
  - 比普通敌人更强
  - 击败后提供构筑能量，不再每只都打断战斗弹出技能选择
  - 击败 5 个后会掉落宝箱（可开商店）

### 开局无敌
- **时间**：游戏开始后 **5秒无敌时间**
- **效果**：免疫所有伤害
- **显示**：屏幕左上角显示倒计时

## 💡 游戏技巧

### 新手建议
1. **选择生存天赋**：额外生命、防护盾和 12% 伤害减免适合熟悉节奏
2. **优先升级防护盾**：提高生存能力
3. **合理使用主动技能**：不要一股脑全放，注意冷却时间
4. **保持移动**：站着不动容易被包围

### 进阶技巧
1. **技能搭配**：
   - 远程角色：优先选择自动攻击、连锁闪电
   - 近战角色：优先选择环绕光球、弹性防护罩
2. **商店购买优先级**：
   - 脆皮角色：优先生命上限和防御力
   - 输出角色：优先减少CD和增加投射物数量
3. **走位技巧**：
   - 利用墙壁阻挡敌人
   - 绕着场地边缘移动
   - 不要被包围，保持移动

### 高级策略
1. **被动技能上限**：最多5个被动技能，选择时要谨慎
2. **金币管理**：击败5个金色Boss后才有商店，提前规划金币使用
3. **Boss战准备**：确保血量充足，技能冷却完毕再挑战Boss
4. **地图重置**：新地图商店升级会重置，需要重新购买

## ❓ 常见问题

### Q1: 游戏启动慢？
**A**: 首次运行需要解压和加载，这是正常现象，等待几秒即可。

### Q2: 杀毒软件报警？
**A**: PyInstaller 打包的程序可能被误报，这是正常现象。可以添加信任或暂时关闭杀毒软件。

### Q3: 中文显示乱码？
**A**: 确保系统安装了中文字体。Windows 系统通常自带，如有问题可安装"微软雅黑"字体。

### Q4: 为什么移动不了？
**A**: 
- 检查是否在输入框中（点击游戏窗口）
- 确保使用 WASD 键，不是方向键
- 中文输入法下 WASD 也能正常工作

### Q5: 商店打不开？
**A**: 
- 需要先击败 5 个金色精英怪
- 拾取掉落的宝箱
- 然后按 **B** 键打开商店

### Q6: 技能选择界面不出现？
**A**: 
- 需要击败精英怪（金色发光的敌人）
- 如果所有技能都满级了，就不会再出现选择界面

### Q7: 游戏卡顿？
**A**: 
- 降低屏幕分辨率
- 关闭其他占用资源的程序
- 游戏本身占用资源较少，一般不会有卡顿

### Q8: 如何保存游戏进度？
**A**: 目前游戏不支持存档功能，每次重新开始都是全新游戏。这是 Roguelike 游戏的特色。

### Q9: 可以修改键位吗？
**A**: 当前版本不支持自定义键位，使用默认键位。

### Q10: 游戏有结局吗？
**A**: 游戏是无限循环的，可以一直挑战新的地图，难度会不断递增。
